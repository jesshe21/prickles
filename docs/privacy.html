Privacy Policy

Last updated: April 15, 2026

Prickles collects, stores, and transmits no personal data. This is the entire policy, but the details are below if you want them.

What we collect

Nothing. There are no analytics, no cookies, no trackers, no session identifiers, no user accounts, and no telemetry. When you visit this page, no identifying information is sent anywhere by Prickles.

What the widget does

The Prickles iOS and macOS widget periodically fetches a single public JSON file from jessica-he.com/prickles/status.json. This file contains only Claude's current status — never anything about you. The widget does not phone home, identify you, or transmit any information about your device.

What the website does

This website uses no cookies, no third-party scripts, and no analytics. It loads a Google Font for typography, which is the only third-party request the page makes. No information about you is sent to Prickles servers (there are no Prickles servers).

Where the data comes from

Claude's status is determined by reading a single public data source on a schedule: Anthropic's public status page at status.anthropic.com. We never fetch anything with your identity attached.

Not affiliated with Anthropic

Prickles is an unofficial, community-built status indicator. It is not affiliated with, endorsed by, or connected to Anthropic PBC. Claude and related marks are trademarks of Anthropic. For authoritative status information, please consult Anthropic's official status page.
